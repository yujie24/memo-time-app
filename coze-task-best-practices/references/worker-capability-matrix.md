# Worker 能力边界矩阵

本矩阵定义了扣子 Worker 的核心能力和限制，用于指导任务分派。

**重要提示**：对于涉及飞书文档、飞书素材的操作，在使用前必须先确认用户是否已安装相关飞书工具，如未安装需提醒用户到"我的-工具"中完成安装后再继续任务。

## 信息获取

### 核心能力
- 联网搜索 (Google/Bing)
- **飞书文档搜索**
- **图片搜索**
- URL 深度解析
- **视觉理解** (原生能力)

### 能力限制
- 无法访问付费墙内容
- 无法登录用户私人账号
- 图片搜索单次最多 10 张，仅辅助创作
- **无需**调用 OCR 工具

### 使用建议
- 在需要搜索时，优先指定搜索关键词和时间范围
- 对于飞书文档，需提供可访问的文档 URL
- 图片搜索仅用于辅助创作，不能替代生图能力

## 内容创作

### 核心能力
- **播客音频生成**
- **微信推文排版**
- **飞书文档编辑**
- **专家文档写作**

### 能力限制
- 播客时长受字数限制 (1000 字≈3 分钟)
- 飞书文档编辑需明确权限
- 专家文档需提供详细任务描述
- **严禁**自行生成 PPT 大纲 (交由专家)

### 使用建议
- 播客任务需明确：音色、脚本风格、字数范围
- 飞书文档编辑需提供：文档 URL、具体修改内容、编辑权限
- 专家文档写作需提供：主题、风格、结构要求、参考资料
- PPT 任务必须调用专家 Agent，不可自行生成大纲

## 视觉设计

### 核心能力
- **Live Photo 生成**
- 文生图/图生图
- 简单的海报排版建议

### 能力限制
- 无法生成可编辑源文件 (PSD/AI)
- 文字渲染能力较弱
- **强制合并调用** (length > 1)
- **严禁**生成数据图表/思维导图

### 使用建议
- 生成多张图片时，必须在单次请求中指定 length > 1，不可循环调用
- 文字需求尽量减少，文字渲染效果有限
- 数据图表必须使用专家 Agent 或专门工具
- Live Photo 生成需明确动态意图描述

## 数据处理

### 核心能力
- **飞书素材上传/下载**
- Python 代码执行 (Sandbox)
- Excel/CSV 读写

### 能力限制
- 沙箱无法联网下载大文件
- 内存限制
- 大文件读取需分批处理

### 使用建议
- 大文件操作需分批处理，避免内存溢出
- Python 执行需注意时间复杂度和内存占用
- 飞书素材操作需提供明确的文件路径和权限

## 逻辑与规划

### 核心能力
- **调用专家 Agent**
- **创建新技能**
- 流程规划与拆解

### 能力限制
- Skill Builder 仅在明确需创建新技能时调用
- Call Agent 需透传原始需求，**严禁**前置加工

### 使用建议
- 调用专家 Agent 时，直接透传用户原始需求，不要过度加工
- 创建新技能时，需明确技能类型、核心功能、使用场景
- 流程规划需明确输入、输出、依赖关系

## 快速查询表

| 能力领域 | 支持任务 | 不支持任务 |
|---------|---------|-----------|
| 信息获取 | 网页搜索、文档搜索、图片搜索 | 付费内容、私人账号访问 |
| 内容创作 | 播客生成、文档编辑、推文排版 | PPT 大纲生成、复杂排版 |
| 视觉设计 | 文生图、图生图、Live Photo | 可编辑源文件、数据图表 |
| 数据处理 | 飞书素材操作、Python 执行、CSV 处理 | 大文件一次性处理 |
| 逻辑规划 | 调用专家、创建技能、流程拆解 | 前置加工专家任务 |

## 注意事项
- 在分派任务前，务必查阅此矩阵确认 Worker 能力
- 避免分配超出能力边界的任务
- 对于"不支持任务"，需转换为"支持任务"或调用专家 Agent
