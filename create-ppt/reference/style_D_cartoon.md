# D. 卡通绘本风

**适用场景**：幼儿园课件、故事绘本、亲子活动、低龄儿童教育

**⚠️ 重要**：此风格**不适用高密度**，幼儿认知有限，每页信息点不宜过多。

**视觉风格速查**：

| 场景关键词 | 视觉风格 | 背景色 |
|-----------|---------|-------|
| 数字/形状/颜色认知、幼儿安全、日常生活 | 糖果色 | 浅黄色 |
| 故事绘本、童话故事、动物认知 | 绘本水彩 | 浅粉色 |

背景色根据PPT主题选定，整份PPT统一使用。

---

## 封面页示例

```
生成一张PPT封面页。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：儿童绘本风格，[浅黄色/浅粉色]背景，糖果色装饰，可爱温馨。

页面中央展示主标题「[标题内容]」，彩色圆润字体，字号小。
主标题下方展示副标题「[副标题内容]」，深棕色，字号更小。

页面配与主题相关的可爱卡通插画，Q版造型。
整体活泼可爱。
```

## 内容页布局

### 单概念认知

```
生成一张幼儿认知绘本页。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：儿童绘本风格，浅黄色背景，糖果色装饰，可爱活泼。

标题「数字3」位于页面顶部，彩色圆润字体。

页面中央展示大大的数字「3」，彩色圆润造型。
数字旁边配3只可爱的卡通小鸟，Q版造型，颜色各异。

页面底部文字：「数一数，有几只小鸟？」，深棕色圆润字体。

周围点缀云朵、星星装饰。
整体简洁可爱，适合幼儿认知。
```

```

页面大面积展示故事场景插画（占60%以上）：
小兔子背着小书包，走在开满花的小路上，远处是彩色的小房子。
画面温馨可爱，水彩手绘质感。

页面底部文字区域：
「小兔子今天第一天上学，它又开心又紧张。」
深棕色圆润字体。

整体温馨童话，适合阅读。
```

## 禁忌

- ❌ 恐怖、暴力、复杂元素
- ❌ 文字过多、字体过小
- ❌ 高密度布局
- ❌ 暗色调、阴森氛围
