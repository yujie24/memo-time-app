# E. 手绘笔记风

**适用场景**：传播导向内容（知识分享、生活常识、产品科普、经验总结），轻松易懂、适合传播

**Prompt结构**：
```
生成一张信息图海报。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：手绘信息图风格，米白色纸张肌理背景，黑色细描边，柔和马克笔平涂。标题的字号小，正文的字号非常小，保持留白充足，搭配简笔图标和符号化小人。

[画面描述]
```

---

## 一、风格定位

手绘信息图与简洁信息图的核心差异：

| 维度 | 简洁信息图（5种风格） | 手绘信息图 |
|-----|---------------------|-----------|
| **画面描述** | 简洁结构，200-350字 | 精细布局，300-500字 |
| **颜色指定** | 不指定，让模型发挥 | 可指定荧光笔等特色元素 |
| **图标使用** | 少量或不描述 | 鼓励详细描述 |
| **特色元素** | 无 | ①②③编号、荧光笔、符号化小人、底部浅黄条 |

---

**适用**：知识分享、生活常识、产品科普、经验总结、读书笔记、方法论


## 三、特色元素

### 标题格式

标题下用荧光笔强调关键词：
- ✅ "高效会议：准备充分，执行有力"，其中"执行有力"下有浅绿色荧光笔强调条
- **同一PPT内荧光笔颜色必须统一**，推荐使用浅绿色

### 布局模式

根据内容结构选择合适的布局：

| 布局 | 适用场景 | 说明 |
|-----|---------|------|
| 多栏并列 | 并列要点、阶段划分 | 三栏/四栏/五栏，用①②③编号分区 |
| 左文右图 | 核心概念+图解 | 左侧文字要点，右侧简笔插画辅助 |
| 金字塔/漏斗 | 层级结构、转化流程 | 从上到下或从大到小的递进 |
| 流程箭头 | 步骤、循环 | 横向或纵向箭头连接 |
| 中心辐射 | 核心+多维度 | 中心概念向外发散 |

### 图标元素

- **简笔图标**：靶心、灯泡、时钟、日历、放大镜、齿轮、书本、勾选、警示、邮件、循环箭头等
- **符号化小人**：小人拿清单、小人在书桌前、团队讨论、导师指导、小人举手等
- **简单图表**：简笔金字塔、简笔漏斗、流程箭头等

### 底部金句

底部浅黄条承载总结性金句：
- ✅ 底部浅黄条「好会议：准时开始，准时结束！」
- ✅ 底部浅黄条「学了就要用，用了才是你的！」

---

## 四、画面描述写作原则（给Agent）

### 鼓励的描述

- **精细布局**：模块化排版，用①②③编号清晰分区，或金字塔/流程箭头等结构
- **图标配对**：靶心图标「明确目的」、时钟图标「准时开始」
- **符号化小人**：小人拿着议程清单、会议桌旁专注的小人们
- **特色元素**：荧光笔强调条、底部浅黄条
- **结构连接**：用箭头连接、形成循环或层级

### 注意事项

- ⚠️ **内容简单时不堆砌图标**：如果内容本身简单，不需要强行给每个点配图标
- ⚠️ **保持手绘感**：描述语言要有手绘、温馨的感觉

### 目标字数

**300-500字**，可比简洁信息图更丰富。
