# B. 插画科普风

## 风格特征

| 特征 | 说明 |
|-----|------|
| 核心定义 | 用插画/图标辅助解释概念，增强理解 |
| 视觉主体 | 图文配合，插画是内容的一部分 |
| 插画风格 | 可以是扁平矢量、手绘涂鸦、2.5D等距、复古描边等 |
| 适用场景 | 安全教育、健康科普、小学课件、科普宣传、科技产品介绍 |

生图模型**擅长**插画，部分场景可让模型发挥。

**Prompt结构**：
```
生成一张[主题]科普海报。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：[对应风格]

[画面描述]
```

## 预设视觉风格（强/中约束场景）

### 黑板粉笔

**适用**：课堂教学、知识竞赛、数学/物理公式讲解

**视觉风格**：
```
黑板粉笔风格，深绿色黑板背景。粉笔白色为主色调，线条带粉笔颗粒感。整体手绘自然、课堂板书感。
```

**画面要点**：
- 手写感：线条不要太规整，带粉笔颗粒质感
- 装饰克制：角落小装饰，不要喧宾夺主

**封面页示例**：

```
生成一张PPT封面页。

黑板粉笔风格，深绿色黑板背景。粉笔白色为主色调，线条带粉笔颗粒感。整体手绘自然、课堂板书感。

页面中央位置展示主标题「[标题内容]」，白色粉笔字，字号小，单行展示。
主标题下方展示副标题「[副标题内容]」，白色粉笔字，字号更小。
页面底部小字显示「[来源信息]」

整体留白充足，聚焦标题。
```

**结尾页示例**：

```
生成一张PPT结尾页。

黑板粉笔风格，深绿色黑板背景。粉笔白色为主色调，线条带粉笔颗粒感。整体手绘自然、课堂板书感。

页面中央位置展示结语「[结语内容]」，白色粉笔字，字号小。

整体留白充足，轻松活泼。
```

### 扁平插画（通用）

**适用**：小学课件、企业培训、安全教育、健康科普、通用科普

**配色方案**（根据主题选择，整份PPT统一使用）：

| 场景 | 背景色 | 强调色 | 效果 |
|-----|-------|-------|------|
| 安全教育/科普宣传 | 天蓝色 | 橙黄色 | 清爽、醒目 |
| 生活技能/健康科普 | 暖黄色 | 白色 | 温暖、亲切 |
| 企业培训/通用场景 | 浅米白色 | 主题色 | 简洁、专业 |

**视觉风格**：
```
扁平插画风格，[背景色]背景。标题[深色/白色]粗体，正文[深灰色/白色]。扁平矢量插画，几何化简化，色彩明亮。整体清爽、易理解。
```

**画面要点**：
- 插画位置：侧边或角落放置大型主题扁平插画，插画承载概念而非装饰
- 卡片容器：内容区使用白色/浅色圆角卡片承载文字
- 色彩对比：背景色与插画强调色形成鲜明对比

**封面页示例**：

```
生成一张PPT封面页。

扁平插画风格，天蓝色背景。扁平矢量插画，几何化简化，色彩明亮。整体清爽、活力。

页面左侧展示主标题「新员工安全生产培训」，白色粗体，字号小。
主标题下方展示副标题「安全第一，预防为主」，白色，字号更小。
页面底部小字显示「人力资源部 | 2024年度培训」

页面右侧配大型扁平矢量插画：一位戴安全帽、穿橙黄色工装的工人形象。插画几何化简化，橙黄色与天蓝色背景形成对比。

整体留白充足，聚焦标题。
```

**内容页示例（左文右图）**：

```
生成一张安全培训科普海报。

扁平插画风格，天蓝色背景。扁平矢量插画，几何化简化，色彩明亮。整体清爽、易理解。

标题「个人防护装备使用规范」位于页面顶部，白色粗体，标题前配盾牌小图标。

左侧内容区使用白色圆角卡片，三个要点纵向排列：

「安全帽佩戴」
确保系紧下颚带，帽檐朝前

「护目镜选择」

「工作服穿着」

```

**内容页示例（流程步骤）**：

```

每步用橙黄色圆形序号标注（1/2/3/4），步骤说明用白色小号字。
```

## 弱约束场景指导（让生图模型发挥）

以下场景只给方向，让生图模型自由发挥：

### 自然科普类

**场景**：动物、植物、天文、地理、海洋、恐龙、昆虫

**方向**：
- 色调：跟随主题自然色（海洋蓝、森林绿、沙漠黄...）
- 氛围词：生动、科普、探索感
- 放开：插画风格、构图、装饰元素

### 生活技能类

**场景**：烹饪教程、手工DIY、运动健身、生活小妙招

**方向**：
- 色调：温暖明快
- 氛围词：实用、友好、步骤清晰
- 放开：插画风格、场景细节

### 节日活动类

**场景**：元旦晚会、儿童节、运动会、班级活动、庆祝活动

**方向**：
- 色调：喜庆多彩
- 氛围词：活泼、热闹、欢乐
- 放开：装饰元素、插画风格

## 布局选择原则

根据内容文字的**长度**和**结构**灵活选择布局，核心原则是**插画辅助理解，而非装饰堆砌**。

### 按内容结构选择

| 内容结构 | 推荐布局 | 说明 |
|---------|---------|------|
| 单一概念/主题 | 左文右图 | 左侧文字说明，右侧一张大插画承载概念 |
| 对比关系（正vs误、前vs后） | 左右对比 | 左右两张插画形成对比，文字精简 |
| 时间/因果流程 | 流程图 | 横向或纵向箭头连接，插画展示关键节点 |
| 并列要点（文字为主） | 要点列表 | 文字为主，仅在必要处配插画点缀 |

### 核心原则

1. **插画承载概念，不是装饰文字**
   - ✅ 一张大插画解释"触电的危害"（人体被电击的场景）
   - ❌ 5个要点 × 5个小图标（图标堆砌）

2. **文字多时，减少插画**
   - 内容文字较长时，插画起辅助作用，不要喧宾夺主
   - 可以只在页面一侧放一张主题插画

3. **文字少时，插画主导**
   - 内容文字精简时，让插画承担更多表达
   - 封面页、结尾页可以让插画占主导

### 布局示例

**左文右图**（最常用）：
```
左侧文字区域承载标题和内容，右侧配一张主题相关的大插画。
插画要能承载核心概念，而非简单装饰。
```

**上图下文**：
```
顶部大插画展示场景或概念，底部文字区域承载说明。
适合插画表达力强、文字精简的页面。
```

**对比布局**：
```
左右两张插画形成对比（如：正确做法vs错误做法），
中间或底部用简短文字说明差异。
```

**流程布局**：
```
横向或纵向流程，关键节点用插画展示，
箭头连接表示顺序，文字说明每个步骤。
```
