# C. 图文混排风

## 风格特征

| 特征 | 说明 |
|-----|------|
| 核心定义 | 实景照片是主要视觉元素，文字仅作简单说明 |
| 视觉主体 | 照片占主导，文字精炼 |
| 适用场景 | 旅游相册、美食摄影、风光摄影、产品展示 |

生图模型**最擅长**自然场景、美食、城市风光，C风格大部分场景可让模型发挥。

---

## 视觉风格写法

**核心原则**：用"载体参照"描述风格，而非描述内容主题。

| 场景 | 视觉风格写法 | 搭配照片 |
|-----|-------------|---------|
| 城市旅游 | 旅行杂志风格 | 城市风光照片 |
| 美食介绍 | 美食杂志风格 | 美食摄影照片 |
| 自然风景 | 风光画册风格 | 自然风光照片 |
| 产品展示 | 产品画册风格 | 产品实物照片 |

## 内容页布局

**布局原则**：
- 根据内容灵活调整，不要照搬模板
- 可使用单图、双图、三图并列、四宫格、九宫格等多种布局
- 视觉风格后添加："标题和正文的字号偏小"

### 左图右文

```
生成一张城市介绍海报。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：旅行杂志风格，白色背景，版式简洁。标题和正文的字号偏小。

页面左侧大图区域展示苏州园林实景照片，亭台楼阁，小桥流水。

页面右侧文字区域：

标题「苏州园林：东方园林艺术的典范」位于右上角。

园林特色
以小见大，咫尺之内再造乾坤
借景造景，移步换景
山水花木，四季皆景

代表园林
拙政园：中国四大名园之首
留园：建筑艺术精华
狮子林：假山迷宫奇观

底部：「上有天堂，下有苏杭」
```
