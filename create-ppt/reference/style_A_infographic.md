# A. 信息图风

**适用场景**：汇报导向内容（工作汇报、政务、正式报告、商务提案、详细调研）

**视觉风格速查**（同一PPT只选择一种视觉风格）：

| 场景关键词 | 视觉风格 |
|-----------|---------|
| 党建/党课/主题党日/表彰大会 | 党政红金 |
| 政务汇报/政策解读 | 政务蓝 |
| 历史/国学/传统文化 | 文化古典 |
| 互联网/科技/创意/教育培训等年轻化行业 | 米白暖色 |
| 传统行业及其他办公场景（默认） | 通用蓝白 |

**Prompt结构**：
```
生成一张信息图海报。

视觉风格（以下内容仅用于指导风格，不要把文字本身写进画面）：[对应风格]

[画面描述]
```

## 一、五种视觉风格

### 通用蓝白（默认）

**适用**：企业汇报、技术分享、产品介绍、架构方案、学术论坛、学术答辩、项目汇报、数据分析、严肃正式的分享场景

**视觉风格**：
```
线性扁平风格，白色工程图纸感的背景，整体呈浅蓝-白色调。标题的字号小，正文的字号非常小，保持留白充足，适当搭配少量扁平的图解元素。不要出现人像。
```

## 二、画面描述写作原则（给Agent）

### 允许的描述

- **布局类型**：流程图、矩阵、左右对比、列表详情等
- **位置关系**：标题位于左上角、左侧占65%、底部总结区域等
- **结构关系**：用箭头连接、分为左右两区域、纵向排列等
- **文字内容**：直接写出要展示的文本（建议150字以上，内容详细具体）
- **简单图解提示**：右侧配简化示意图、配架构图等

### 不允许的描述

- ❌ **正文指定颜色**：橙色高亮、绿色增长 — 视觉风格已约束整体色调，正文颜色让模型自由发挥
- ❌ **具体字号数值**：36px、24号字 — 用描述性词汇（字号小、字号更小）
- ❌ **大量图标描述**：每个要点配XX图标 — 让模型自由发挥
- ❌ **复杂图表**：柱状图、饼图、折线图 — 生图模型不擅长

> 注：标题颜色可以写，各风格有固定标题颜色（通用蓝白用黑色/深蓝色，党政红金用暗红色等）

### 目标字数

**200-350字**，其中实际展示的文字内容建议**150字以上**，鼓励内容详细具体。

## 三、布局类型速查（给Agent）

| 布局类型 | 适用场景 | 结构要点 |
|---------|---------|---------|
| **流程图** | 步骤说明、演进过程、因果关系 | 横向/纵向步骤，用箭头连接 |
| **矩阵(2x2)** | 分类、对比、多维度分析 | 四个象限等分 |
| **左右对比** | A vs B、问题vs方案 | 左右分栏，中间VS分隔 |
| **列表详情** | 产品介绍、方案详情、特性说明 | 左侧列表为主，右侧图解为辅 |
| **主从聚焦** | 核心KPI展示（慎用） | 核心数据居中，辅助数据左右排列 |
| **漏斗** | 转化漏斗、筛选过程 | 从上到下逐层变窄 |

> **主从聚焦布局**仅在需要突出单一关键数据时使用，不作为常规布局推荐。

## 四、封面页与结尾页

> 各风格的封面页示例见上方对应章节。以下为通用结构参考。

**封面页结构**：
- 开头：`生成一张PPT封面页。` + 视觉风格描述
- 主标题：页面中央位置，字号小，单行展示
- 副标题：主标题下方，字号更小
- 底部信息：日期/作者/机构
- 结尾：`整体留白充足，聚焦标题。`

**结尾页结构**：
- 开头：`生成一张PPT结尾页。` + 视觉风格描述
- 中央：感谢语或总结金句
- 底部：联系方式（可选）
- 结尾：`整体简洁，大量留白。`

## 五、内容分析指导（给Agent）

### 如何识别内容类型

拿到用户需求后，先判断核心是什么类型：

```
├─ 有步骤/流程关系？ → 流程图
├─ 有2-4个并列分类？ → 矩阵 / 并列卡片
├─ 有A vs B对比？ → 左右对比
├─ 有详细特性/要点列表？ → 列表详情
├─ 有核心数据需要突出？ → 主从聚焦
├─ 有转化/筛选过程？ → 漏斗
└─ 纯观点/原则？ → 要点列表
```

### 图解描述原则

描述图解时，**简单提及即可**，不需要过度详细：

| ❌ 过度描述 | ✅ 简洁描述 |
|-----------|-----------|
| "右侧是一个IDE界面简化示意图，深蓝色线条风格，左侧代码编辑区，右侧AI对话窗口，两者融合为一体，标注一体化体验" | "右侧配IDE界面简化示意图" |
| "用天平图表示ROE与估值的平衡关系，左侧托盘是ROE，右侧托盘是估值" | "右侧配天平示意图，表示平衡关系" |

让生图模型根据内容自由发挥图解细节。

### 风格选择指南

| 场景特征 | 推荐风格 |
|---------|---------|
| 传统行业、企业内部汇报、技术分享、学术答辩 | 通用蓝白 |
| 互联网/科技/创意/教育培训等年轻化行业 | 米白暖色 |
| 政府部门、事业单位工作 | 政务蓝 |
| 党建活动、党课培训 | 党政红金 |
| 历史文化、国学经典、传统文化 | 文化古典 |
